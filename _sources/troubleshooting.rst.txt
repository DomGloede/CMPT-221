Troubleshooting
===============
This section describes some common issues that can arise and possible solutions.

Issue #1
--------
Cannot create a virtual environment
-----------------------------------
Ensure that you are cd into the correct directory before attempting to create the virtual environment.

Issue #2
--------
Cannot run app.py
-----------------------------------
Ensure that you have installed all the necessary packages and libraries via the requirements.txt file. 
If you are still having issues running the command ``pip install -r requirements.txt`` in the terminal,
ensure that you are in the correct direcotory containing requirements.txt.

Issue #3
--------
Cannot run tests
-----------------------------------
Ensure that you are in the correct directory containing the test file you wish to run. 
Also ensure that you have installed all the necessary packages and libraries to run the tests.
